Our work focuses on DevOps platform engineering, GitHub automation, and Azure infrastructure, helping teams improve developer workflows, automate CI/CD pipelines, and implement scalable engineering practices across the software delivery lifecycle.

## Kernel51

Kernel51 is a microkernel-based orchestration engine designed to power modular automation and platform integrations. Instead of tightly coupled tooling, Kernel51 provides a pluggable runtime that allows automation capabilities to be added as independent modules. This architecture enables teams to build flexible automation systems that evolve over time without introducing unnecessary complexity.

The **krnl51 CLI** provides the command-line interface for interacting with the Kernel51 runtime, enabling engineers to execute workflows, manage modules, and integrate automation into modern development and DevOps environments.
